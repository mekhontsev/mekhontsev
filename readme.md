## Scientific Results

### [The aspect ratio of the Twin Dragon is 1/φ](papers/twin-dragon/) *(2026)*

[![DOI](https://zenodo.org/badge/DOI/10.5281/zenodo.19443672.svg)](https://doi.org/10.5281/zenodo.19443672)

The Twin Dragon fractal is defined purely via the Gaussian integer $1+i$,
yet its geometric aspect ratio equals $1/\varphi$, where $\varphi=(1+\sqrt{5})/2$
is the golden ratio. Proved by explicitly solving the covariance fixed-point
equation for the self-similar measure.

📄 [PDF](papers/twin-dragon/twin-dragon-aspect-ratio.pdf) · [arXiv:2604.05010](https://arxiv.org/abs/2604.05010)

## Software

### [IFStile](https://github.com/mekhontsev/ifstile) *(GPL, C++/C)*

A free cross-platform application for the discovery, visualization, and
classification of self-similar sets and tessellations. Features:

- Multidimensional IFS construction in arbitrary dimensions (full 2D/3D support)
- Automated fractal discovery: rep-tiles, carpets, dragons, quasicrystal tilings
- Dimensional analysis: Hausdorff dimension via analytical and numerical methods
- Geometric analysis: centroids, moments, aspect ratios of fractal sets
- High-resolution rendering and keyframe animation export
- Declarative domain-specific language [AIFS](https://github.com/mekhontsev/ifstile/blob/main/manual.md) with JavaScript integration

Runs on Windows, macOS, Linux, Android, and WebAssembly.

🌐 [ifstile.com](https://ifstile.com) · [Run in browser](https://app.ifstile.com)

### [IFS Encyclopedia](https://ifsdb.github.io)

A static reference site cataloguing iterated function systems — definitions,
AIFS programs, interactive fractal renderings, and bibliographic references.
Built with Astro and rendered client-side via `ifslib.wasm`.
